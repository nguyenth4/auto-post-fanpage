Đăng nhanh (TikTok/YouTube)

An toàn/tiện nhất là dùng OAuth. Với YouTube: chọn file video và upload (resumable upload). Với TikTok: cần video_url public và app có quyền Content Posting API.

Yêu cầu token có scope youtube.upload. Khuyến nghị connect bằng Google OAuth để có refresh token.

Kết nối Facebook bằng Token

Dán Fanpage ID và Page Access Token. Hệ thống sẽ kiểm tra token và lưu page vào danh sách kết nối (multi-page).

Lấy token tại Graph API Explorer hoặc Access Token Tool. Quyền thường cần: pages_show_list, pages_read_engagement, pages_manage_posts.

Kết nối TikTok bằng Token

Nhập Open ID và Access Token. Hệ thống sẽ kiểm tra token qua TikTok API và lưu vào danh sách kết nối.

Lưu ý: để đăng video qua API cần bật quyền theo TikTok app của bạn (Content Posting API).

Kết nối YouTube bằng Token

Nhập Channel ID và Access Token (Google). Hệ thống sẽ kiểm tra channel và lưu vào danh sách kết nối.

Token phải có scope upload (ví dụ: https://www.googleapis.com/auth/youtube.upload) nếu muốn đăng video.
